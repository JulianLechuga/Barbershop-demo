# Barbershop-demo

Modern and premium barbershop website demo.

## Features
- **Premium Design**: Dark theme with gold accents.
- **Responsive**: Mobile-first approach.
- **Automated Deployment**: GitHub Actions integration for GitHub Pages.

## Tech Stack
- React 18
- TypeScript
- Vite
- Framer Motion
- Lucide React

## Local Development
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
